## Kot-039
Fie o funcție (implementata ca o corutina) de sumare a trei variabile să se scrie un program Kotlin care aplică curry și uncurry în cazul apelului direct cu un set de valori.

## Pyt-080
Utilizând funcții din itertools și un generator creat de programator să se genereze automat nume de fișiere temporare care vor fi create astfel s1-index-s2.tmp unde s1 și s2 sunt nume primite la apelul metodei. Metoda va primi, verifica unde a rămas generatorul și va întoarce următorul nume.
